# Exemples Terraform

Ce dossier contiendra des exemples simples d’utilisation des modules Terraform.

## Objectif

Les exemples permettront de tester rapidement un module sans dépendre d’une configuration complète d’environnement.

Exemples futurs :

```text
examples/
├── simple-resource-group/
├── simple-storage-account/
└── simple-network/
```

## Bonnes pratiques

- Garder les exemples simples
- Ajouter un README par exemple
- Fournir un fichier `terraform.tfvars.example`
- Prévoir une section de nettoyage
